> **__Guide__**
.tag:guide
1) Drink an Extreme Magic Potion (or better) before going in with prayers on
    • Prayers should be Protect from Melee + Augury
2) Spam Target Cycle + <:Vuln:537349530551582720> + <:sonic:535533809924571136> (or <:dbreath:535533833391702017>) while going into spot displayed in image below
    • The spot is 1 tile Southwest of the middle mushshroom
.
.img:https://i.imgur.com/jlN3dUY.png
.
3) Cast Sunshine <:sunshine:994921119381463070> after you have targeted Helwyr <:Helwyr:513213019635843100>, then fellow the **General approach**
	a) If he Mushroom gases your <:sunshine:994921119381463070> area at the start, teleport out and restart

**__General Approach__**
⬥ While in <:sunshine:994921119381463070> aim to do 1 <:wm:535533809978966037>, 1 <:asphyx:535533833072672778>, and 1 <:gstaff:513203008608141314> <:spec:537340400273195028>
⬥ Use Resonance <:res:535541258844635148> to heal when <:Helwyr:513213019635843100> Cleaves (ideally without Protect melee on)
⬥ Use Devotion <:devo:513190158728953857> or Debilitate <:debil:535541278264393729> when <:Helwyr:513213019635843100> does his Frenzy
    • Alternatively, run away if <:devo:513190158728953857> is on cooldown
    • Use Freedom <:freedom:535541258240786434> after the Frenzy
⬥ Use <:gstaff:513203008608141314> <:spec:537340400273195028> whenever possible (costs ~23 adrenaline with <:vigour:615613235512737792>)
    • Ensure you still have adrenaline for <:devo:513190158728953857> etc. when you need it (needs >50% adrenaline)
⬥ Avoid moving <:Helwyr:513213019635843100> unless absolutely necessary as he skips autos when he moves
    • Only scenarios you should move <:Helwyr:513213019635843100> are:
        - Walking <:comb:535533833098100745>
        - To avoid the Frenzy when <:devo:513190158728953857> is on cooldown
        - To avoid the mushroom

.
**__Tips__**
⬥ When trying to time Target Cycle, it is *important* to run while you spam it, otherwise you cannot spam Target Cycle
    • If you Target Cycle properly, <:Helwyr:513213019635843100>'s HP box will not show
⬥ You can only Devotion <:devo:513190158728953857> every other Frenzy
⬥ Whenever you cast <:devo:513190158728953857> / <:freedom:535541258240786434>, try to weave in an <:AirSurge:543465115870035999> or <:Vuln:537349530551582720> with it
    • Be aware, this will not work after <:gconc:869285393223254107> / <:conc:535533833106489365> / <:asphyx:535533833072672778>
    • If using 2H, input <:AirSurge:543465115870035999> / <:Vuln:537349530551582720> at the same time as your next ability
    • If using DW, input <:AirSurge:543465115870035999> / <:Vuln:537349530551582720> at the same time as <:devo:513190158728953857> / <:freedom:535541258240786434>
