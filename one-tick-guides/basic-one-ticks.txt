## __K'ril Tsutsaroth, General Graardor, Commander Zilyana and King Black Dragon__
.tag:Intro
This guide will cover the basic strategy to get your first 1 tick boss kills. These 1 tick kills are fairly consistent, and are a good entry point to make progress towards minimum tick feats roles.

⬥ These Bosses have been grouped together due to them using the same setup and rotation. This works as they are in a similar HP range (40-60k), K'ril and Zilyana will need better luck or more granite mauls due to having slightly more hp.
    • You may have to incorporate a <:gmaul:796989663051120640> <:eofspec:1257438999794946099> end with <:ingen:641339234111848463>
⬥ For general 1 tick kill information, and some useful tips and tricks make sure to read <#1249793025996099708>.

.

### Rotation
This is the basic rotation for these speed kills, many other rotations can be done instead.

**War's Retreat Build**
<:dba:603979368850653216> <:spec:537340400273195028> + equip <:masterwork2hsword:1274330676061278334> → Use the crystal til 120% adren → Enter portal → Join Instance

**In Instance**
⬥ At any point after entering the instance switch to Grim <:grim:568262896375824385> and activate.
⬥ The four basics before chaos roar can be replaced with any basic.

<:anti:535541306475151390> + <:surge:535533810004262912> → <:freedom:535541258240786434> + drop <:dummy:656844963522281473> → <:slice:535532879594258432> → <:sever:535532879577612298> → <:chaosroar:994644356671737966> (<:gfury:535532879334080527> if not owned) → s <:cane:535532878969438210> → ( tc → <:smokecloud:856635090641879050> + r <:cane:535532878969438210> → <:gmaul:796989663051120640> <:eofspec:1257438999794946099> → <:gmaul:796989663051120640> <:eofspec:1257438999794946099> → <:enhreplen:634350450887622656> → <:gmaul:796989663051120640> <:eofspec:1257438999794946099> → <:deci:535532879325822986> / <:backhandflank:867678153854025779>)

### Rotation
This is the basic rotation for these speed kills, many other rotations can be done instead.

**War's Retreat Build**
Build 12 <:necrosis:1139452791878856805> stacks with the following rotation:
<:touchofdeath:1137809175980810380> → <:livingdeath:1159434908486357072> → <:touchofdeath:1137809175980810380> → <:necroauto:1137809137401602109> → <:necroauto:1137809137401602109>
Off and wait 10 seconds to drop combat:
Equip <:excal:641337999170207763> + (tc) + <:deathguard90:1138809243143766118> <:eofspec:1257438999794946099> + click floor

**In Instance**
⬥ At any point after entering the portal switch to <:grim:568262896375824385> and activate.
⬥ You only need the second rotation including <:gmaul:796989663051120640> <:eofspec:1257438999794946099> for Commander Zilyana.
⬥ <:dragonslayer:641339921814126594> / <:demonslayer:641339921675845633> can be used for extra damage on KBD / K'ril after <:deathguard90:1138809243143766118>

Wait until <:deathguard90:1138809243143766118> <:eofspec:1257438999794946099> has 10 seconds or less on its cooldown then enter the instance:
Go through door + <:dive:1049378668197195808> → wait until <:dive:1049378668197195808> has 11 seconds on its cooldown remaining (1s if mobile is on your gear) →
⬥ ( tc + <:smokecloud:856635090641879050> + <:deathguard90:1138809243143766118> <:eofspec:1257438999794946099> )
⬥ ( tc + <:smokecloud:856635090641879050> + <:deathguard90:1138809243143766118> <:eofspec:1257438999794946099> + <:zekkil:903244090953588787> + <:ingen:641339234111848463> + <:gmaul:796989663051120640> <:eofspec:1257438999794946099>)
