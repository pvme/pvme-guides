# Rasial, the First Necromancer

## __Introduction__
.tag:intro
In order to fight Rasial you must complete the [Alpha vs Omega](<https://runescape.wiki/w/Alpha_vs_Omega>) quest. Rasial deals Necromancy and typeless damage, and can currently only be killed using Necromancy.

.
## __Positioning__
.tag:positioning
.img:https://img.pvme.io/images/z4ifLrUqck.png
⬥ The tile shown above serves as a safespot for the ghost wall mechanic.
    • This tile is ideal as it keeps you 5 tiles away for quicker <:deathskulls:1159434663903899728>
    • For the soul obelisks, you may run 4 tiles west (toward obelisk), and 2-4 east.
        ⬩ Doing this allows <:deathskulls:1159434663903899728> to kill one of the soul obelisks.
        ⬩ This is only recommended if you have the ruthless perk.
⬥ For an in-game example of the correct tile being used, [click here](<https://img.pvme.io/images/N8u6ViKvDL.mp4>).

.tag:reqs

⬥ Undead damage boosts <:salveamulet:797899945730244648> <:undeadslayer:641339922019516416> <:undeadslayerperk:689502804720615441>
⬥ <:igneouskalzuk:902209626479685734> / <:igneouskalmor:1139200072438005851> for <:deathskulls:1159434663903899728>
⬥ <:conservationofenergy:697808773711921195> Conservation of Energy + <:furyofthesmall:697808773917573233> Fury of the Small (Equilibrium Aura) **or** <:berserkersfury:697808774106185768> Berserker's Fury (Invigorate Aura).
⬥ <:invig4:712073087859949570> perk.
⬥ <:reaverpouch:839903693837959228> auto fire rate set to 3-7.
    • Spam eat <:blueblubber:689530593742291033> under <:splitsoul:1137809168368148490> if not using <:reaverscroll:1022562484378419284>
⬥ The Lunar spellbook is used for <:veng:543478434953822208>
⬥ All of the Necromancy talent tree unlocked.
    • Greater bone shield <:greaterboneshield:1137809118283976814> active.
