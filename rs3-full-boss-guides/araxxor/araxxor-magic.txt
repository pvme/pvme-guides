**__Preset Notes__**

⬥ It is recommended to have the *NopeNopeNope* perk from Player Owned Farms active.

⬥ Upkeep the <:kalgscroll:841409588954923049> buff from your familiar.

⬥ Araxxor takes a few seconds to clear vulns on phase 1-3, making vuln bombs <:vulnbomb:655341074235129858> usable on all phases.

.
> **__Top Path__**
.tag:top
⬥ **Phase 1**
    • This phase is timegated so it can be used to set yourself up for p2 while you deal damage
    • Stall <:tsunamiincite:958989794757730307> when the web burn reaches ~80%
    • Release with <:gconc:869285393223254107> then build and <:nat:535541258131865633> at close to 100% web burn
    • <:gconc:869285393223254107> then build adren till web burns completely
    • You may wait for Araxxor to do a special attack after web finishes burning before going into phase 2 (not needed but provides safer skips)

.
⬥ **Phase 2**
_This phase is skipped_
Turn towards web, and when it burns, drink the <:accel:756236265472524418> and <:surge:535533810004262912> → <:bd:535532854281764884> → <:surge:535533810004262912> → <:surge:535533810004262912> → <:bd:535532854281764884> straight to phase 3, then immediately drop a <:dummy:656844963522281473>

.
⬥ **Phase 3**
While waiting for Araxxor to come up the path do the following (use filler abilities as needed)
<:kalgscroll:841409588954923049> + <:gsunshine:994644352871714836> → <:fsoa:869284271595069451> → <:HydrixBakri:550834403136503822> <:gricocaroming:867678153966878740> → s <:omnipowerigneous:1179505745654448208>

When Araxxor is within range of your <:gsunshine:994644352871714836>, swap your autocast spell to <:icebarrage:537340400185245701> and do the following
r <:omnipowerigneous:1179505745654448208> (disassemble your <:dummy:656844963522281473>) → <:smoketendrils:536257336130404372> → <:wm:535533809978966037> → if Araxxor heals or if this fails to phase him, then improvise

.
> **__Middle Path__**
.tag:mid
⬥ **Phase 1**
    • Walk Araxxor so he is right against the web (see below how to walk p1)
    • At 99% web burn you can bladed dive <:bd:535532854281764884> + surge <:surge:535533810004262912> into phase 2
        ⬩ You'll know if you did it correctly if Araxxor starts absorbing the acid at 1:06 rather than the standard 1:12. This can be delayed if rax does a spec right before the web finishes burning.
    • Below is a diagram to help with positioning Araxxor on P1 (Camera facing east)
.
.img:https://img.pvme.io/images/KtRYknU.jpg
.
        ⬩ The yellow line is the line to stand on after walking Araxxor, if you move 1 more square east Araxxor will run into the web and start going west again which could ruin the setup. If you have trouble finding this spot just keep in mind you must stand 1 east of the line where the skulls are (circled in red).
    • Here is a video example on walking Araxxor in P1:
.
https://youtu.be/A_6ah7GF_1k
.
⬥ **Phase 2**
    • Run across the acid pool and 3 tiles onto the ramp. This puts Araxxor as far up the path as he can be while still absorbing acid.
.
.img:https://img.pvme.io/images/U49xswa.jpg
.
    • Deal damage
    • When rax has absorbed 53% acid, you can <:bd:535532854281764884> + <:surge:535533810004262912> up the ramp, then walk 1 tile west so you have room to dodge cleave specials at the top of the ramp.
        ⬩ You'll know you did this correctly if rax goes up to 55% then starts dropping, and you'll get the popup message that the bridge is burning when there is 49% acid left. This is the perfect amount needed.
    • Set up for phase 3
        ⬩ Natural instinct <:nat:535541258131865633> timing is calculated based on how much acid rax has poured onto the bridge. The timing is roughly when the bridge has absorbed **25-30%** acid. Under normal circumstances, this means the popup would appear at **49%**, so you would <:nat:535541258131865633> when the bar says **24-19%**.
    • <:tsunamiincite:958989794757730307> and <:fsoa:869284271595069451> <:spec:537340400273195028> then build to 100%

.
⬥ **Phase 3**
(tc) <:Araxxi:513213019543699466> + <:vulnbomb:655341074235129858> + <:kalgscroll:841409588954923049> + <:meta:535533811304497183> → <:gconc:869285393223254107> → <:dbreath:535533833391702017> → <:omnipowerigneous:1179505745654448208> → <:smoketendrils:536257336130404372> → <:wm:535533809978966037>

.
> **__Bottom Path__**
.tag:bot
⬥ **Phase 1**
    • No special setup. Just DPS Araxxor down to <5k hp.

.
⬥ **Phase 2**
    • No special setup. Follow the light and wait for the charge. Stay at 100% adrenaline.
    • When you dodge the charge and Araxxor damages the wall, you can click the minimap to start running into phase 3, and then right click Araxxor when he is going up into the ceiling so you can right click 'attack' instantly in phase 3. This is nice in case minions spawn during p2 to avoid Target Cycle targeting them instead of Araxxor.

.
⬥ **Phase 3**
<:kalgscroll:841409588954923049> + <:meta:535533811304497183> + <:adrenrenewal:736298121704767538> → <:smokecloud:856635090641879050> <:sonicwave:1087373972443775056> → <:gconc:869285393223254107> → <:dbreath:535533833391702017> → <:icebarrage:537340400185245701> <:tsunamiincite:958989794757730307> → <:fsoa:869284271595069451> <:spec:537340400273195028> → <:gconc:869285393223254107> → <:omnipowerigneous:1179505745654448208> → <:smoketendrils:536257336130404372>

.
