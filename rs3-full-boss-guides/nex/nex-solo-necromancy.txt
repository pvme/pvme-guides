## __Rotation__
.tag:rotations
### Pre-fight
<:conjurearmy:1166094935066423348> + <:veng:543478434953822208> + walk to middle-south → <:lifetransfer:1137809128136388819> → <:commandghost:1159434409913634816> → <:invokedeath:1137809121983336548> → (at around 4s) <:dealerlegs80:1142783488257429504> + <:anti:535541306475151390> → <:firstnecrorobebottom:1206664822654115911> + <:splitsoul:1137809168368148490> → <:commandskeleton:1137809194423160883>
### Smoke Phase
(tc) + <:vulnbomb:655341074235129858> + <:deathskulls:1159434663903899728> + walk 1 tile north-east of Nex → <:soulsap:1137809140476031057> → <:touchofdeath:1137809175980810380> + <:dive:1049378668197195808> to Fumus
*Note: stalling crackling between kills can help make this phase more consistent.*

**Fumus**
(tc) + <:vulnbomb:655341074235129858> + <:bloat:1159433682403201044> → <:bbc:626714879218155521> + <:grico:787904334812807238> → <:omniguard:1138809234922934282> + <:soulsap:1137809140476031057> → walk towards nex for skull bounce
*Note: you are aiming to have skulls bounce onto Fumus and then back to Nex.*
.
### Shadow Phase
<:omniguard:1138809234922934282> <:spec:537340400273195028> → <:deathguard90:1138809243143766118> <:eofspec:1257438999794946099> → <:deathspark:1143642264850141205> <:necroauto:1137809137401602109> if not 120k + <:surge:535533810004262912> + <:dive:1049378668197195808> to Umbra
*Note: The (tc) timing is when <:bloat:1159433682403201044> kills Fumus as this will then tag Nex with a hit of <:splitsoul:1137809168368148490> . This allows the <:omniguard:1138809234922934282> <:spec:537340400273195028> to not be nulled.*

**Umbra**

(tc) Umbra + <:vulnbomb:655341074235129858> + <:bloat:1159433682403201044> → <:bbc:626714879218155521> + <:grico:787904334812807238> → <:omniguard:1138809234922934282> + <:soulsap:1137809140476031057>
.
### Blood Phase
<:vulnbomb:655341074235129858> + <:necroauto:1137809137401602109> → <:divert:787904334377648130> → <:prep:535541258546970624> → <:commandskeleton:1137809194423160883> → <:bloat:1159433682403201044> → <:fingerofdeath:1159434801938432010> + <:adrenrenewal:736298121704767538> → <:soulsap:1137809140476031057> → <:touchofdeath:1137809175980810380>

*Note: Move towards Cruor during this phase.*

**Cruor**

(tc) Cruor + <:bloat:1159433682403201044> → <:bbc:626714879218155521> + <:grico:787904334812807238> → <:omniguard:1138809234922934282> + <:soulsap:1137809140476031057> → <:necroauto:1137809137401602109> if not dead

*Notes:*
⬥ *If you do not get a darkness skip, ensure you are attacked 3 times to start the Ice phase on Nex's 'Contain this!' special attack.*
⬥ *Position Nex near Cruor if possible so that <:bloat:1159433682403201044> spreads to her when Cruor dies.*
.
