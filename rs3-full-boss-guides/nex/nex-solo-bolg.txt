> **__Introduction__**
.tag:intro
When Nex spawns for the first time in a custom instance it will be praying Deflect Magic <:DeflectMage:544195487926845462> for the majority of the kill, this is why Range only kills are the meta for fast consistent kills. After every kill you will teleport out of the instance and re-enter in order to force Nex to always be praying Deflect Mage <:DeflectMage:544195487926845462>

*Note: It is strongly advised to complete the **Ritual of the Mahjarrat** quest in order to get access to the Jas boost which gives a 5% damage boost in GWD1 (including Nex). This can be done via Fairy ring code D.K.Q.*

> **__Rotation with ECB EoF (~34-35 KPH)__**
.tag:bolgecbeof
**Useful tips**
⬥ Make sure to use Jas boost <:jasboost:1006121672790773770>
⬥ Keep last listed arrow equiped until a new arrow is specified
⬥ Stay in combat between kills to stall crackling, allowing it to proc in p1 for a more consistent phasing
⬥ Wait 1t after using grico on a minion before tagging Nex to allow ecb to hit

**__Pre-Fight__**
<:anticlearheaded:867678154071998464> + <:vulnbomb:655341074235129858> right as Nex says "Fill my soul with smoke!" (~9s) → <:ecb:615618531937222657> <:eofspec:746403211908481184>

*Note: If you can phase Nex into blood before she does "Embrace darkness", you can start with Sacrifice rather than Blood siphon and save time.*
